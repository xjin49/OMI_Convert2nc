A python package to convert that converts OMI HCHO (TEMIS or QA4ECV retrieval) data or OMI NO2 (QA4ECV retrieval) from ASCII format to NetCDF.
Daily OMI HCHO (TEMIS retrieval) data can be downloaded at: http://h2co.aeronomie.be/ch2o/ch2o.php?instr=omi&period=day
Monthly average OMI HCHO (QA4ECV retrieval) data can be downloaded at: http://www.qa4ecv.eu/ecv/hcho-p/data
Monthly average OMI NO2 (QA4ECV retrieval) data can be downloaded at: http://www.qa4ecv.eu/ecv/no2-pre/data
